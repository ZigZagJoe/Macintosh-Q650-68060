# Smartsocket
This contains data related to the 040 to 060 "Smartsocket" adapters.

It must be responsible for:
* mechanical socket adaptation
* feed 3.3v (not 5v) to the 060
* mux the IPL lines for configuration strap
* implementating the 1/2 busclock mode (cpu runs 2X bus).
